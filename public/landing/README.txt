Put your landing images here, then set the matching path in src/lib/content.ts

Examples:
  public/landing/coach.jpg              → hero.coachPhoto: "/landing/coach.jpg"
  public/landing/hero/rahul.jpg         → heroTransforms[0].image: "/landing/hero/rahul.jpg"
  public/landing/transforms/t1-before.jpg → transformations[0].before: "/landing/transforms/t1-before.jpg"
  public/landing/transforms/t1-after.jpg  → transformations[0].after: "/landing/transforms/t1-after.jpg"

Use .jpg / .png / .webp. Prefer portrait photos for before/after cards.
