**Upper/Lower/Push/Pull 4-Day Split + Active Recovery**

You've completed your foundation phase with solid upper/lower work. This week we're shifting to a 4-day push/pull/upper/lower rotation to increase training frequency per muscle group while keeping session length around 60 minutes. Volume is increased by ~15% through added working sets. Given your low energy (4/10) and 5-hour sleep average, we're prioritizing compound movements and minimizing excessive isolation to respect recovery capacity.

**Weekly Structure:**

**Day 1 – Push (Chest/Shoulders/Triceps)**
- Barbell Bench Press: 4 sets x 6–8 reps, 3 min rest
- Dumbbell Incline Press: 3 sets x 8–10 reps, 2 min rest
- Cable Lateral Raise: 3 sets x 12–15 reps, 90 sec rest
- Overhead Dumbbell Press: 3 sets x 8–10 reps, 2 min rest
- Cable Tricep Pushdown: 3 sets x 12–15 reps, 90 sec rest

**Day 2 – Pull (Back/Biceps)**
- Barbell Bent-Over Row: 4 sets x 6–8 reps, 3 min rest
- Lat Pulldown (Cable): 3 sets x 8–10 reps, 2 min rest
- Dumbbell Single-Arm Row: 3 sets x 10–12 reps per arm, 90 sec rest
- Face Pull (Cable): 3 sets x 15–20 reps, 90 sec rest
- Dumbbell Hammer Curl: 3 sets x 10–12 reps, 90 sec rest

**Day 3 – Lower (Quad Focus)**
- Barbell Back Squat: 4 sets x 6–8 reps, 3 min rest
- Leg Press: 3 sets x 10–12 reps, 2 min rest
- Dumbbell Walking Lunge: 3 sets x 8–10 reps per leg, 2 min rest
- Leg Curl (Machine): 3 sets x 12–15 reps, 90 sec rest
- Standing Calf Raise: 3 sets x 15–20 reps, 90 sec rest

**Day 4 – Upper (Full Upper Body)**
- Dumbbell Bench Press: 4 sets x 8–10 reps, 2 min rest
- Cable Seated Row: 4 sets x 8–10 reps, 2 min rest
- Dumbbell Shoulder Press: 3 sets x 10–12 reps, 90 sec rest
- Cable Lat Pulldown (Wide Grip): 3 sets x 10–12 reps, 90 sec rest
- Push-Ups: 2 sets x AMRAP, 2 min rest

**Day 5 – Active Recovery / Tempo Run**
- 30–40 min tempo run at conversational pace OR brisk walk + mobility work (hip flexor stretch, thoracic rotations, cat-cow, ankle circles)

**Rest Days:** 2 full rest days per week. Prioritize 7–8 hours sleep, hydration (2L+), and stress management.

**Progression Notes:**
- Aim to add 1–2 reps or 2.5–5 kg to compound lifts each week when form and energy allow.
- If energy remains low, reduce working sets by 1 per exercise and focus on movement quality.
- Track sleep and hunger daily; address sleep hygiene (dark room, consistent bedtime, limit screen time 1 hour before bed) to improve recovery and reduce cravings.

**Cardio Integration:**
- 2 x 30 min brisk walks on rest days (low intensity, recovery-focused)
- 1 x tempo run on Day 5 (moderate intensity, supports athletic performance goal)

**Key Focus:**
- Compound movements prioritized to maximize efficiency and hormonal response
- Moderate rep ranges (6–15) to balance strength and hypertrophy
- Rest periods calibrated to allow adequate recovery between sets
- Volume increased by 15% from foundation phase via additional working sets
- Session duration remains ~60 minutes per training day
