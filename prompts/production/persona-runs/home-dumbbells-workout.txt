Meera Home's muscle gain limited-equipment home program (Intermediate level). 3 training days per week with progressive overload. Warm-up: 5–8 min light cardio plus dynamic mobility (hip circles, band pull-aparts, arm swings). Cool-down: 5 min stretching. Rest days: 8k+ steps and light mobility for recovery. Prioritize form, RIR 2–3 on compounds, add weight when all sets are clean.
Monday — Upper (dumbbells)
  • DB bench press 4x8
  • DB row 3x10
  • DB lateral raise 3x12
  Note: Start each session with the warm-up block above; finish with cool-down stretching.

Wednesday — Lower (dumbbells)
  • Goblet squat 4x10
  • DB Romanian deadlift 3x10
  • DB lunges 3x10
  Note: Start each session with the warm-up block above; finish with cool-down stretching.

Friday — Full body
  • DB shoulder press 3x10
  • DB curl 3x12
  • Plank 3x45s
  Note: Start each session with the warm-up block above; finish with cool-down stretching.
