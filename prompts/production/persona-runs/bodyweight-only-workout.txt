Ananya BW's fat loss bodyweight program (Intermediate level). 3 training days per week with progressive overload. Warm-up: 5–8 min light cardio plus dynamic mobility (hip circles, band pull-aparts, arm swings). Cool-down: 5 min stretching. Rest days: 8k+ steps and light mobility for recovery. Prioritize form, RIR 2–3 on compounds, add weight when all sets are clean.
Monday — Upper bodyweight
  • Push-ups 4x12
  • Pike push-ups 3x10
  • Plank 3x45s
  Note: Start each session with the warm-up block above; finish with cool-down stretching.

Wednesday — Lower bodyweight
  • Bodyweight squats 4x15
  • Walking lunges 3x12
  • Glute bridge 3x15
  Note: Start each session with the warm-up block above; finish with cool-down stretching.

Friday — Full body
  • Split squats 3x10
  • Inverted rows 3x12
  • Mountain climbers 3x20
  Note: Start each session with the warm-up block above; finish with cool-down stretching.
