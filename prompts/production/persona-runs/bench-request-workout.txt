Vikram Bench's muscle gain home barbell program (Intermediate level). 3 training days per week with progressive overload. Warm-up: 5–8 min light cardio plus dynamic mobility (hip circles, band pull-aparts, arm swings). Cool-down: 5 min stretching. Rest days: 8k+ steps and light mobility for recovery. Prioritize form, RIR 2–3 on compounds, add weight when all sets are clean.
Monday — Upper
  • Barbell bench press 4x8
  • DB row 3x10
  • Push-ups 3x12
  Note: Start each session with the warm-up block above; finish with cool-down stretching.

Wednesday — Lower
  • Goblet squat 4x10
  • Romanian deadlift 3x10
  • Walking lunges 3x12
  Note: Start each session with the warm-up block above; finish with cool-down stretching.

Friday — Full body
  • Overhead press 3x8
  • Band pull-apart 3x15
  • Plank 3x45s
  Note: Start each session with the warm-up block above; finish with cool-down stretching.
